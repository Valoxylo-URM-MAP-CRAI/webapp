# VALOBOIS — Documentation

Documentation for VALOBOIS, the web tool for **evaluating reclaimed / second-hand wood**
("bois d'occasion"). Written for diagnosticians (end users) and the people maintaining the tool.

## Start here

- **[How the assessment works](methodologie-evaluation.md)** — the evaluation methodology: how your
  ratings become a value profile and an **orientation** recommendation (réemploi / réutilisation /
  recyclage / combustion). This is the core of the tool.
- **[Glossary](glossaire.md)** — the domain vocabulary (French ↔ English) in plain language.

## Feature-by-feature

The per-feature specifications live in **[`../specs/`](../specs/README.md)** — one document per
feature (authentication, the evaluation library, each editor tab, the exports, bilingual support,
reference data), each describing what the feature does for the user.

## Project principles

The documentation principles (as-built, domain-first, plain language) are recorded in the project
constitution at [`../.specify/memory/constitution.md`](../.specify/memory/constitution.md).
