# Backups projet VALOBOIS

Ce dossier centralise les snapshots utiles pour diagnostiquer ou corriger des regressions.

## Archives disponibles

- `archive-2026-05-27` : regroupement des backups utilises pendant la remise en oeuvre de la logique Etiqueter (separation lot/piece, disponibilite, UI).

## Convention

- Un dossier par date de consolidation: `archive-YYYY-MM-DD`.
- Conserver un `README.md` dans chaque archive avec:
  - contexte fonctionnel,
  - liste des fichiers,
  - empreintes SHA256,
  - procedure de restauration.
